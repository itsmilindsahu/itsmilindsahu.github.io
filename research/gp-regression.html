Overview

Project Summary

This internship project replicates and extends the simulation study from Bachoc et al. (2018), which introduces a Gaussian Process regression framework where inputs are probability distributions rather than scalar or vector-valued points.

KEY INSIGHT: When input data consists of distributions (e.g., histograms, empirical measures), the appropriate geometry is the Wasserstein-2 metric on the space of probability measures — not Euclidean distance. A power-exponential kernel built on W₂ defines valid, positive-definite covariance functions over distribution-valued inputs.

Wasserstein-2 Distance for 1D Distributions

For one-dimensional probability distributions, the Wasserstein-2 distance reduces to the L² distance between quantile (inverse CDF) functions. This enables efficient, closed-form computation compared to the general OT problem:

# For empirical distributions with quantile functions F⁻¹_μ and F⁻¹_ν
# on a common grid of quantile levels t ∈ [0,1]:
W2_sq = np.mean((F_inv_mu - F_inv_nu) ** 2)
W2    = np.sqrt(W2_sq)

Power-Exponential Kernel on W₂

The covariance function is a power-exponential (Matérn family) kernel parameterized by the W₂ metric:

# Parameters: sigma2 (variance), theta (length-scale), alpha (smoothness ∈ (0,2])
# eta (nugget for numerical stability)
def kernel(mu_i, mu_j, sigma2, theta, alpha):
    d = wasserstein2(mu_i, mu_j)
    return sigma2 * np.exp(-(d / theta) ** alpha)

# Full covariance matrix with nugget:
K = [[kernel(mu_i, mu_j, sigma2, theta, alpha) for mu_j in X]
     for mu_i in X]
K += eta * np.eye(n)   # nugget term
Simulation Study

Replication of Bachoc et al. (2018) — Table I

The simulation replicates the main experiment from the paper: 100 training distributions drawn from a GP-perturbed distribution family, with 500 test distributions. Three methods are compared on mean squared prediction error (MSPE) and 95% coverage.

Table: MSPE and 95% coverage — GP-Wasserstein kernel vs. Legendre and PCA baselines (n=100 train, n=500 test, 100 simulation runs). Lower MSPE is better; coverage target is 0.95.

Method Setting MSPE ↓ 95% Coverage Notes
GP-Wasserstein α=1 (Matérn-like) 0.041 0.948 Correct geometry; best overall
GP-Wasserstein α=2 (Gaussian) 0.038 0.951 Smooth functions, better fit
Legendre-k5 k=5 coefficients 0.127 0.891 Undercoverage; lossy projection
Legendre-k10 k=10 coefficients 0.089 0.912 Improved with more features

PCA-k10 k=10 components 0.076 0.921 Closer but still lossy

The Wasserstein GP consistently outperforms both projection baselines across all settings, confirming that using the correct metric geometry for distribution-valued inputs provides material gains in prediction accuracy and calibration.

Methods

Technical Approach

MLE Parameter Estimation

Hyperparameters (σ², θ, η) are estimated via maximum marginal likelihood. Multi-start L-BFGS-B optimization (8 restarts from random initial points) avoids local minima. The nugget η ensures numerical stability of the covariance matrix inversion via Cholesky decomposition.

def neg_log_likelihood(params, X_quants, y):
    sigma2, theta, alpha, eta = params
    K = build_cov_matrix(X_quants, sigma2, theta, alpha, eta)
    L = np.linalg.cholesky(K)
    alpha_vec = cho_solve((L, True), y)
    nll = 0.5 * (y @ alpha_vec + 2*np.sum(np.log(np.diag(L))))
    return nll

Kriging Prediction

Given observed distribution-valued inputs and scalar outputs, the GP posterior provides the Best Linear Unbiased Predictor (BLUP) with analytical uncertainty quantification:

# Predictive mean and variance at new distribution input μ*
k_star = [kernel(mu_star, mu_i, sigma2, theta, alpha) for mu_i in X_train]
y_pred = k_star @ K_inv @ y_train
var_pred = kernel(mu_star, mu_star, ...) - k_star @ K_inv @ k_star

Baselines

Legendre Projection — Each distribution is represented by its first k Legendre polynomial coefficients of its quantile function, reducing the problem to standard GP regression in ℝᵏ. Fails to capture distributional geometry beyond truncated linear projections.

PCA Projection — Projects empirical quantile functions onto the top k PCA components learned from training data. More adaptive but still discards the metric structure of the Wasserstein space.
