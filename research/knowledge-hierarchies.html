Abstract

The Central Hypothesis

This research investigates a striking mathematical connection between two independently developed theories from completely different fields: Garicano's (2000) knowledge hierarchy model from organizational economics, and Telgarsky's (2016) neural depth theorem from machine learning theory.

CONJECTURE: Both theories — organizational depth optimization and neural network depth selection — reduce to the same KKT inequality under a unified Lagrangian framework, implying they are formal analogs of the same optimization problem. The optimal depth in each domain is determined by the same mathematical condition expressed in different notation.

Both sides reduce to the same structure: a threshold condition on a "complexity" parameter that determines the optimal depth. The Lagrange multiplier λ plays the role of the wage differential in Garicano's model and the approximation tolerance in Telgarsky's.

Lagrangian Optimization & KKT Threshold Solver

A Python formulation solving the dual organizational/neural optimization problem and verifying the KKT boundary conditions numerically:

import numpy as np

def solve_optimal_depth(complexity_kappa, cost_c, lambda_multiplier):
    """
    Finds the optimal depth d* that minimizes total resource cost 
    subject to the complexity constraint.
    
    L(d, lambda) = cost_c * d + lambda_multiplier * (complexity_kappa / d - 1)
    """
    # KKT threshold condition check
    # Stationarity: dL/dd = cost_c - lambda_multiplier * complexity_kappa / d^2 = 0
    d_continuous = np.sqrt((lambda_multiplier * complexity_kappa) / cost_c)
    
    # Primal feasibility and integer constraint check
    d_candidates = [max(1, int(np.floor(d_continuous))), int(np.ceil(d_continuous))]
    costs = [cost_c * d + lambda_multiplier * (complexity_kappa / d) for d in d_candidates]
    d_optimal = d_candidates[np.argmin(costs)]
    
    # Verify KKT complementary slackness
    slack = complexity_kappa / d_optimal - 1
    is_kkt_valid = abs(lambda_multiplier * slack) < 1e-5 or lambda_multiplier == 0
    
    return d_optimal, is_kkt_valid
Comparison

Two Theories, One Structure

Economics · Garicano (2000)
Knowledge Hierarchy Model
Organizations are optimal knowledge hierarchies. Workers solve problems up to some complexity threshold z*; exceptions escalate to managers. The optimal depth L* satisfies a threshold condition on problem frequency p(z) and knowledge costs c, determined by first-order KKT conditions of a total-cost minimization problem.
ML Theory · Telgarsky (2016)
Neural Depth Theorem
Deep networks efficiently represent functions that shallow networks require exponentially more neurons to approximate. The minimum network depth d* for approximating a function f at tolerance ε satisfies an analogous complexity threshold — a KKT-like condition on the target function's complexity class C(f).

IMPLICATION: If the formal equivalence holds, the optimal organizational depth for any task is directly predictable from the complexity of the AI model required for that task — providing a rigorous foundation for AI-augmentation of knowledge hierarchies.

Research Questions

Open Problems

Q1 · COMMON COMPLEXITY MEASURE
Can we define a single "complexity" measure C(·) such that both organizational depth and neural depth are determined by the same threshold condition C(problem) ≥ κ*?
This requires identifying the appropriate notion of problem complexity shared between an organizational task (Garicano's framework) and a function approximation task (Telgarsky's). Candidate: the VC dimension of the problem-solving rule, or the Lipschitz constant of the optimal decision boundary.
Q2 · IDENTICAL COMPARATIVE STATICS
Do the optimal depth responses to parameter changes follow the same monotone comparative statics in both models?
Milgrom-Shannon monotone comparative statics theory predicts that if both models share a common Lagrangian, their depth responses to changes in environment parameters (task complexity, communication cost, approximation tolerance) must be monotone and identical in sign. This is the key falsifiable prediction.
Q3 · AI-AUGMENTED ORGANIZATIONS
If the analogy holds, what does it imply for optimal organizational design when AI handles a subset of problem complexities?
If an AI system can solve problems up to complexity z_AI < z*, the organizational hierarchy can be "flattened" by exactly the number of levels corresponding to this complexity reduction. This gives a precise, theory-grounded answer to the question: "How many management layers should AI eliminate?"
